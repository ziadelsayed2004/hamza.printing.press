# Visual RTL Material QA Checklist

لكل خطوة UI:

- [ ] لا يوجد `style={{...}}` جديد.
- [ ] لا يوجد `!important` جديد.
- [ ] لا يوجد `sx={{...}}` كبير أو عشوائي.
- [ ] كل نص عربي محاذي يمين.
- [ ] كل label فوق الحقل وبمسافة سليمة.
- [ ] كل field full width داخل grid cell.
- [ ] drawer له header/content/footer موحد.
- [ ] sidebar على اليمين ولا يغطي المحتوى.
- [ ] topbar لا يكسر الـ layout.
- [ ] light mode واضح.
- [ ] dark mode واضح.
- [ ] mobile/tablet/desktop مقبول.
- [ ] لا يوجد horizontal overflow.
- [ ] لا توجد هوية قديمة.
