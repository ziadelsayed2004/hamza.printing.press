# 110 — Inventory Receipts/Sales/Returns Reconciliation

## الهدف
اضبط المخزون والواردات والمبيعات والمسترجعات.

المطلوب:
- receipts تزيد المخزون.
- invoice sale تخصم المخزون.
- return يزيد المخزون.
- adjustment بصلاحية وسبب.
- لا تعديل صامت للstock.
- stock reports وalerts.

قبول الخطوة:
- stock current = sum ledger transactions.
- كل حركة لها reference وaudit.
