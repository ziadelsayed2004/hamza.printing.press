# 085 — Supply Remittance Finance UI

## Goal
Create a professional finance UI for pending, collected, supplied, and unsupplied balances.

## Scope
- Finance dashboard cards:
  - الرصيد المعلق.
  - المحصل الفعلي.
  - المحصل والمورد.
  - المحصل وغير المورد.
- Payment history table with supply status badges.
- Action to mark collected payments as supplied for authorized users.
- Outlet statement page.
- Filters by outlet, date, supply status, collection status.
- EGP/Egypt timezone formatting.

## Acceptance
- Admin/accountant can identify exactly what is collected and not supplied.
- Outlet statement reconciles with invoice/payment ledger.
- No installment panels/cards remain.
