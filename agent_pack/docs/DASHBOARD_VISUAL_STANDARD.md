# Dashboard Visual Standard

## Direction
Professional Arabic RTL Material UI dashboard with premium light/dark modes. No legacy identity, no cheap admin-template look, no overuse of dialogs.

## Layout
- Clean topbar with account menu, theme toggle, global search/action area.
- Sidebar grouped by business domain.
- Responsive content width with balanced cards and spacing.
- Breadcrumbs only where helpful.

## Visual language
- Neutral professional palette.
- Strong readability.
- Subtle elevations.
- Consistent border radius.
- Calm icons.
- Clear active/hover states.
- Elegant empty states for fresh database.

## Dashboard content
- KPI cards.
- Quick actions.
- Recent activity.
- Low stock / overdue / pending panels.
- Reports preview.
- Empty-state onboarding when no records exist.

## UX rules
- Major create/edit flows should be pages or drawers, not chained dialogs.
- Confirmation dialogs only for destructive or irreversible actions.
- Tables must use consistent filters, search, pagination, row actions, loading, and empty states.
- Forms must show validation and clear submit states.
