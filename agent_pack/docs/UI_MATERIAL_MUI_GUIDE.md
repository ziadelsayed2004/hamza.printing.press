# Material UI Direction

## Visual goal
Professional Google-like Material dashboard, Arabic RTL, responsive, fast, clean and not popup-heavy.

## UI rules

- Use a persistent dashboard layout.
- Use sidebar + topbar + breadcrumbs.
- Use MUI cards, tables, chips, drawers, forms, tabs.
- Use DataGrid for complex lists.
- Use dialogs only for confirmations and small quick actions.
- Use pages or drawers for complex create/edit flows.
- Use Snackbar/Toast for feedback.
- Use skeleton/loading states.
- Use empty states.
- Use permission-aware navigation.
